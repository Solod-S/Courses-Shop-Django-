===============================
ДОБАВЛЯЕМ ЗАПИСИ В БАЗУ
===============================

1. Входим в Django shell:

py -m manage shell

2. Импортируем модели:

from shop.models import Category, Course

3. Смотрим все записи:

Course.objects.all()      (получаем все курсы)
Category.objects.all()    (получаем все категории)

4. Создаем новую категорию:

new_category = Category(title="Programming")
new_category.save()

5. Проверяем созданную запись:

new_category.id           (получаем id созданной категории)
new_category.created_at   (получаем дату создания)

6. Ищем запись по primary key:

Category.objects.get(pk=1)
Category.objects.get(pk=1).title

7. Фильтруем записи по названию:

Category.objects.filter(title="Programming")
Category.objects.filter(title="Programming")[0].title


8. Подготавливаем категорию для создания курса:
category = Category.objects.get(id=1)
category.course_set.all()    (получаем все курсы этой категории)


9. Создаем новый курс в этой категории:
category.course_set.create(title="Complete Python Guide", price=9.99, students_qty=100, reviews_qty=50)


10. Проверяем созданный курс:
Course.objects.get(pk=1)    (получаем курс по primary key)


11. Получаем список названий всех курсов:
[course.title for course in Course.objects.all()]
